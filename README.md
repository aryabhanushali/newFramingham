# Beyond Framingham: Predicting CVD Risk from Wearable Data

Predicting 10-year cardiovascular disease risk using only 
passively collected wearable sensor data from NHANES 2011-2012,
without requiring blood draws or clinical visits.

## Project Structure
- `data/` — NHANES XPT files (not tracked by git)
- `notebooks/` — EDA and modelling notebooks
- `src/` — reusable Python modules
- `results/` — plots and outputs

## Data
Download from: https://wwwn.cdc.gov/nchs/nhanes/continuousnhanes/default.aspx?BeginYear=2011

Files needed: DEMO_G, TCHOL_G, HDL_G, BPX_G, SMQ_G, 
GLU_G, GHB_G, DIQ_G, PAXHD_G, PAXDAY_G, PAXHR_G
